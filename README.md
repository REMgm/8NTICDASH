# 8NTICDASH v2.0 — Control Center

**8NTIC Autonomous AI Agency — Real-Time Command Hub**

Dynamic dashboard with live GitHub integration, animated statistics, and real-time agent monitoring.

## v2.0 Features

- **Live GitHub Integration** — Auto-fetches commits every 60 seconds
- **Real-Time Clock** — Live day counter from Genesis, ticking seconds
- **Enhanced Animations** — Smooth transitions, skeleton loaders, particle effects
- **Dynamic Charts** — Auto-updating commit activity graphs
- **Live Agent Status** — Online/idle/offline with pulse indicators
- **Mission Tracking** — Progress bars with glow animations
- **Dark/Light Mode** — Seamless theme switching

## Live Data

- **Day Counter:** Auto-calculates days since Genesis (March 6, 2026)
- **GitHub API:** Fetches real commits from REMgm/8NTIC
- **Agent Status:** Live online indicators
- **Auto-Refresh:** Polls every 60 seconds

## Stack

- React 18 (CDN)
- Chart.js 4.4.1
- GitHub REST API
- Pure CSS animations
